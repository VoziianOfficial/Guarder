Access comparison

Visitor entry Compare guest codes, call boxes, and remote access.
Gate type Review compatibility with swing, slide, or existing gates.

User permissions

User permissions can help manage family, guests, workers, property managers, or service providers. Compare permission levels, admin controls, and temporary access options.

Admin controls Review who can add, edit, or remove users.
Temporary users Compare limited-time and one-time entry permissions.
Role settings Ask whether users can have different access levels.

Entry schedules

Entry schedules can support recurring visitor access, worker access windows, household routines, and limited-time permissions. Compare schedule flexibility before choosing a package.

Recurring schedules Useful for weekly service access or routines.
Expiration rules Review automatic code expiration controls.
Time windows Compare day, hour, or custom access windows.

Access logs

Access logs may show when doors, gates, or entry points were used and by which credential. Compare activity history, app visibility, alert options, and data retention.

Activity history Review what entry events are stored.
User visibility Compare whether access is tied to specific users.
Alert options Ask about real-time notifications for entry events.
